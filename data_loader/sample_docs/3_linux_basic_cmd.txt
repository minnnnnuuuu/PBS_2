[문서 제목: 신입 엔지니어를 위한 리눅스 필수 명령어]

서버 관리를 위해 반드시 알아야 할 리눅스 명령어 리스트입니다.

1. 파일 및 디렉토리 관리
   - ls: 현재 디렉토리의 파일 목록을 보여줍니다. (옵션: -al 숨김 파일 포함 상세 보기)
   - cd [경로]: 해당 경로로 이동합니다.
   - mkdir [폴더명]: 새로운 디렉토리를 만듭니다.
   - rm -rf [폴더명]: 디렉토리와 그 안의 파일을 강제로 삭제합니다. (주의해서 사용)

2. 시스템 상태 확인
   - top: 실행 중인 프로세스와 CPU/메모리 점유율을 실시간으로 보여줍니다.
   - df -h: 디스크 용량을 사람이 읽기 쉬운 단위(GB, MB)로 보여줍니다.
   - free -m: 메모리 사용량을 MB 단위로 확인합니다.

3. 네트워크 확인
   - ping [주소]: 해당 서버와 통신이 되는지 확인합니다.
   - netstat -tuln: 현재 열려있는(Listening) 포트 번호를 확인합니다.
